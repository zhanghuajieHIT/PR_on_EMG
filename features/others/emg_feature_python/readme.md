## Introduction

该计算库主要用来提取肌电信号的特征.


计算的特征主要包含以下4个部分

1.  统计特征
2.  时域特征
3.  频域特征
4.  时间-频率特征

## 使用 

Git clone之后, 可直接作为一个module进行调用


## 关于第三方库的说明

本库依赖于下面四个第三方库:

1.  [numpy] 
2.  [scipy] 科学计算库并提供信号处理接口
3.  [nitime] 时序数据分析库
4.  [pywavelets] 小波变换库

上述四个依赖库都可以通过 easy_install 进行自动安装，如要安装nitime，在命令行中输入：
  easy_install nitime
